Open issues over time

Points before today are reconstructed from issue timestamps — open count at month m = issues created before m minus issues closed before m. Points from the weekly collector are exact live snapshots. Toggle labels in the legend; hover the chart for values.

Lines changed per month

Volume of change flowing into flang/ + flang-rt/, from git log --numstat. Rising commits / lines / contributors mean development is accelerating; falling means it is cooling off. Dashed verticals mark LLVM X.Y.0 releases. Contributors are unique author emails per month.

Total lines of code in flang/ + flang-rt/ of llvm/llvm-project, counted with cloc from a sparse checkout each run. Points are exact snapshots; the series builds forward from the first collection. Historical backfill is intended as a one-time offline run, not CI.

Data collected weekly via GitHub Actions · counts from the GitHub REST API. Not affiliated with the LLVM project.
